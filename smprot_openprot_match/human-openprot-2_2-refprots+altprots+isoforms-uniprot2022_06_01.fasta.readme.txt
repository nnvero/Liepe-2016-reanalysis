### Protein FASTA Sequence Header Lines

General format:

```
>Identifier|TX=TaxonomyIdentifier OS=OrganismName GN=GeneName TA= TranscriptAccession PA=ProteinAccession
```

##### Where: 

+ Identifier Proteins annotated in OpenProt have an accession number starting either with IP (for Inferred Protein, i.e. predicted AltProt) or with II (Inferred Isoform, i.e. novel predicted isoform of RefProt). Protein accession numbers for RefProts originate from Ensembl, NCBI RefSeq or UniProtKB annotations. If the "RefProts Included" option is selected, the FASTA file (protein) will contain all RefProt sequences (non-redundant protein sequences from UniProtKB / SwissProt, Ensembl and NCBI RefSeq annotations). When more than one protein accession number is available for one sequence, UniProtKB annotation is used as the Identifier and others are included in the PA field.


+ TX Taxonomy Identifier.

+ OS Organism Name.

+ GN Gene Name.

+ TA Transcript accession number(s) (from Ensembl and/or NCBI RefSeq annotations).

+ PA Protein accession number(s).

### Difference between RefProts, AltProts and Isoforms 

+ RefProts are proteins translated from annotated CDSs found in transcripts classified as mRNAs.

+ AltProts are predicted proteins translated from alternative ORFs (AltORFs) found in transcripts classified as mRNAs or from unannotated AltORFs found in transcripts classified as non-coding RNAs (ncRNAs). Within mRNAs, an altORF may be present in the UTRs or overlapping the annotated CDS in a frameshifted reading frame.

+ Isoforms are novel predicted proteins translated from alternative ORFs (AltORFs) found in transcripts classified as mRNAs or from unannotated AltORFs found in transcripts classified as non-coding RNAs (ncRNAs) that share a clear sequence homology with a RefProt from the same gene.

### File Naming Scheme

```
The files are consistently named following this pattern:
    <species>-openprot-<version>-<refprots AND/OR altprots and isoforms>-<supporting evidence (MS) (IF ALL -> "all")>-<annotation>-<uniprot YES/NO (IF NO -> EMPTY)>.fasta.zip
    <species>-openprot-<version>-<refprots AND/OR altprots and isoforms>-<supporting evidence (MS) (IF ALL -> "all")>-<annotation>-<uniprot YES/NO (IF NO -> EMPTY)>.tsv.zip

<species>:       The systematic name of the species.
<version> :      OpenProt release version
<refprot AND/OR altprot and isoforms> : Reference and/or alternative proteins
<supporting evidence (MS)>: Mass spectrometry detection evidence of alternative proteins and isoforms newly annotated in OpenProt
<annotations>: Genome annotations

fasta (protein) : All files in these directories are FASTA database files with protein sequences
fasta (dna) : All files in these directories are FASTA database files with DNA sequences
tsv: All files in these directories are tsv database files
```
