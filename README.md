# Liepe-2016-reanalysis

## Data

Download the dataset from [Dryad](https://datadryad.org/dataset/doi:10.5061/dryad.r984n)
and unzip it under [`data/`](data/)

the structure should look like
```
data/
    raw/
        C1R/*
        immunopeptidome/*
        lysate/*
        T2/*
    mzml/
        GR_LCL/*
    mylonas-et-al/supplementary6.xlsx
```
